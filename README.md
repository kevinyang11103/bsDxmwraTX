# Python图书管理系统（2025） python382

## 项目概述

本项目是基于Django框架开发的Python图书管理系统，旨在提供便捷的图书管理功能。系统使用Python 3.9及以上版本（建议3.11），后端数据库采用Mysql，支持通过Navicat或Dbeaver进行管理。

## 技术栈

- Python版本：3.9及以上（建议3.11）
- 开发框架：Django
- 开发软件：Pycharm
- 数据库：Mysql
- 数据库软件：Navicat/Dbeaver

## 功能模块

### 管理页功能

- **图书管理**：实现对图书信息的增、删、改、查操作。
- **出版社管理**：对出版社信息进行维护。
- **作者管理**：管理作者信息。
- **数据统计**：使用Echarts进行数据可视化展示。
- **登录注册**：用户认证功能。
- **修改密码**：用户密码管理。

## 系统角色

- 系统管理员：负责整个图书管理系统的维护。
- 普通用户：进行图书查询、借阅等操作。

## 运行环境

- Python 3.9及以上版本
- Django框架
- Mysql数据库
- Navicat或Dbeaver（可选）

## 部署步骤

1. 安装Python 3.9及以上版本。
2. 安装Mysql数据库。
3. 使用Pycharm等开发工具克隆本项目。
4. 配置数据库连接。
5. 运行Django项目。

## 目录结构

```plaintext
project/
│
├── apps/               # 应用目录
├── config/             # 配置文件目录
├── db/                 # 数据库相关文件
├── media/              # 媒体文件目录
├── static/             # 静态文件目录
├── templates/          # 模板目录
└── manage.py           # 管理脚本
```

## 核心亮点

- 使用Django框架，遵循MVC设计模式，易于维护和扩展。
- 数据统计模块采用Echarts，提供直观的数据可视化。
- 系统设计考虑用户友好性，易于操作使用。
